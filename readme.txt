Создание сниппетов с ценами в поисковой системе Yandex


== Description ==

Создание сниппетов с ценами в поисковой системе Yandex
Плагин формирует YML файлы для страниц и записей которые необходимо добавить в Яндекс.Вебмастер

Примеры использования:

Если на сайте представлена страница или запись с описанием услуги, которая имеет свою стоимость, с помощью данного плагина можно вывести эту цену сразу в поисковую выдачу Яндекса, что повышает CTR сниппета
На странице настройки плагина представлен скриншот сниппета.

Внимание! Плагин разрабатывался для собственных нужд и распространяется "как есть", если вам необходим дополнительный функционал, или же любые другие услуги, пишите на почту info@new-webstudio.ru


== Installation ==

1. Разархивируйте плагин в папку "/wp-content/plugins/" или установите через стандартную установку в разделе "Плагины"
2. Активируйте плагин в разделе "Плагины"
